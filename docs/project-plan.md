# Project Plan

## Goal

Build a resume-ready AI pharmacovigilance product that demonstrates drug safety data analysis, real-world public API integration, dashboard design, and AI-assisted scientific reporting.

## MVP Scope

1. Query openFDA FAERS by drug name.
2. Aggregate top adverse reactions and seriousness.
3. Display reviewer-oriented visual summaries.
4. Generate a structured safety report with FAERS limitations.
5. Export the report for portfolio evidence.

## Next Milestones

Detailed milestones are tracked in [roadmap.md](roadmap.md).

The highest-priority next improvements are:

1. Add source cards that show exact openFDA query assumptions and request URLs.
2. Add disproportionality metrics such as PRR and ROR.
3. Add drug-vs-drug comparison for same-class products.
4. Add report persistence and history.
5. Add PDF export.
6. Add curated test cases for common drugs.

## Resume Framing

Built an AI-powered pharmacovigilance dashboard using openFDA FAERS data to analyze adverse event patterns, serious outcomes, demographic distributions, and safety signals for marketed drugs. Integrated LLM-based structured report generation with source-grounded limitations and reviewer-ready summaries.
